# 📊 Mean-Variance-Standard Deviation Calculator

> A Python project built using **NumPy** to calculate statistical measures for a **3×3 matrix**.
>
> This project is part of the **freeCodeCamp Scientific Computing with Python Certification**.

---

## 📸 Project Preview

```
Input:
[0,1,2,3,4,5,6,7,8]

Matrix:

0 1 2
3 4 5
6 7 8
```

The calculator returns:

- ✅ Mean
- ✅ Variance
- ✅ Standard Deviation
- ✅ Maximum
- ✅ Minimum
- ✅ Sum

for

- Columns
- Rows
- Entire Matrix

---

# 🚀 Features

✔ Convert a list into a 3×3 NumPy array

✔ Calculate Mean

✔ Calculate Variance

✔ Calculate Standard Deviation

✔ Find Maximum values

✔ Find Minimum values

✔ Calculate Sum

✔ Input Validation

✔ Raises ValueError if input length is not 9

